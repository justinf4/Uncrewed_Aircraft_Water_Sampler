# Local Libraries

Place downloaded library zip files here, extracted as individual folders.

PlatformIO auto-discovers libraries placed in this directory — no extra config needed.

See `../docs/libraries.md` for download links and instructions.

## Expected structure after downloading

```
lib/
├── ESP32Servo/
├── TinyGPSPlus/
├── DS18B20/
└── OneWire/
```
